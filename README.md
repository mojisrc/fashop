## 最新功能

- 自定义后台拖拽界页面；
- 默认3种商品分类页面；
- 订单管理；
- 退款退货管理；
- 收藏列表；
- 地址管理，1.用户添加，2.调用微信地址管理接口；
- 购物车管理，支持切换规格；
- 商品管理，支持多规格切换

## 命令行介绍
```bash
php fashop build fashop-v1.0.1
// build后跟fashop-v会自动生成压缩包fashop-v1.0.1.tar.gz，由于github远程下载比较卡，所以升级包是放在国内的服务器，这个命令是方便创建生产环境的压缩包。
```
